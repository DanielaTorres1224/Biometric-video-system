Sistema de Biometría con Reconocimiento Facial
Sistema de control de acceso mediante reconocimiento facial utilizando tecnologías web modernas.
Características

Reconocimiento facial en tiempo real
Captura de video desde la cámara web
Interfaz moderna y responsive
Sistema de autenticación biométrica
Registro de usuarios con validación facial

Tecnologías

React
Vite
TailwindCSS
JavaScript

Requisitos

Node.js versión 14 o superior
npm o yarn
Cámara web funcional
Navegador moderno

Instalación

Clona el repositorio:

bashgit clone https://github.com/DanielaTorres1224/Biometric-video-system.git

Navega al directorio:

bashcd Biometric-video-system

Instala las dependencias:

bashnpm install

Inicia el servidor:

bashnpm run dev

Abre http://localhost:5173 en tu navegador

Uso
El sistema permite registrar usuarios mediante reconocimiento facial y posteriormente autenticar su identidad comparando el rostro capturado con los registros almacenados.
